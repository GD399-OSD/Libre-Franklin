FONTLOG for the Libre Franklin fonts

This file provides detailed information on the Libre Franklin Font Software.

This information should be distributed along with the Libre Franklin fonts and any
derivative works.

Basic Font Information

Libre Franklin is an 1912 ATF Franklin Gothic revival
